# Hoverse - Ứng dụng khám phá địa điểm cho Gen Z

Dự án Full-Stack (Spring Boot + ReactJS) giúp Gen Z khám phá các địa điểm vui chơi, hidden gems tại TP.HCM, kết hợp đề xuất thông minh từ AI (Gemini).

## 🚀 Công nghệ sử dụng

- **Backend:** Java 17, Spring Boot 3.x, Spring Data JPA, Spring Security
- **Frontend:** ReactJS, Axios, React Router
- **Database:** MariaDB
- **AI Integration:** Google Gemini API
- **Khác:** JWT Authentication, Cloudinary/AWS S3

## 📌 Tính năng cốt lõi (MVP)

- [ ] Random discovery & Filter địa điểm.
- [ ] Đăng tải địa điểm mới (Drop pin) - Validate chống trùng lặp.
- [ ] AI Recommendation - Gợi ý địa điểm theo ngữ cảnh.
- [ ] Tương tác xã hội cơ bản (Review, Save).

## 🏗️ Kiến trúc hệ thống

- Mô hình: Client - Server (RESTful API).
- Cấu trúc Backend: Controller - Service - Repository.
